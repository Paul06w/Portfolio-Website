DECEMBER 2023 - JUNE 2024

MBot-Project

The Mbot project employed an agile approach based on Scrum and iterative sprint planning. The goal of the project was to enable the Mbot to be controlled via a Java Spring Boot server and a web interface. In addition to manual control, the Mbot was to be equipped with a line-following algorithm to significantly enhance the robot's navigation capabilities. This flexible methodology supported incremental development.
